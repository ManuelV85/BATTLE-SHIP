# BattleShip game

>Battleship is a strategy type guessing game for two players. It is played on ruled grids on which each playerÂ´s fleet of warships are maked.The locations of the fleets are concealed from the other player. Players alternate turns calling "shots" at the other player's ships, and the objective of the game is to **destroy the opposing player's fleet.**

## Installation

>You can run this repository following the next steps:

- Clone this repository from (https://github.com/ManuelV85/BATTLE-SHIP.git)
- Install the proyect dependencies by running ***npm install***
- Start the server by running ***npm run start***
